Standing on others shoulders to get a little higher - Node.js.

Open Source

Developed and maintained by an open source community at GitHub. Feel free to fork the repository and send us a pull request with some cool improvements.

Decoupled WebApps

Web apps are great but there are a lot of pieces to keep track of and development set-ups can get complicated or tedious. Hem tries to help simplify by handling things like building your production resource files to hadling proxys to an API durring development

TDD!! - Real Devices

Utilizing jasmine test syntax (sprinkled with some coffeescript magic if you like) and testacular you can replace a lot of web-driver/selenium types of tests with fast running tests that run on real browsers and real devices such as phones, tablets or headless PhantomJS.

Easy Debugging

Hem leverages tools that can save you lots of typing. If you are used to pure js and css there can be some new stuff to learn (if you want to...) however hem reduces the pain since syntax errors get caught and reported as soon as they appear.

Spine, Jasmine, Stylus, Jade

Hem is built to be used for development of web-apps using a specific set of libraries. There is some flexibility, however, it really makes using these chosen tools together uber smooth!

Coffeescript?!

There is a certain philosophy behind Coffeescript. Without trying to detail whatever it is in words we can say some like it, and with hem we're trying to bring some of that to this level of tooling.
